# Oblique Strategy

Get a random Oblique Strategy card printed into your console whenever you need inspiration.

Run `npx oblique-strategy` to get a strategy card logged into your terminal

## Examples

```
┌────────────────────────┐
│                        │
│  Remove a restriction  │
│                        │
│    - Edition 4 (1996)  │
│                        │
└────────────────────────┘
```
